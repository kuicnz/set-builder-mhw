# set-builder-mhw

When Monster Hunter: World was released in 2018, there were few good build planners available, so I created one for myself. It supports both English and Chinese. Later, as more advanced community tools became available, I stopped maintaining this project.

《怪物猎人：世界》于 2018 年发布时，网上几乎没有好用的配装工具，因此我开发了这款应用。它支持英文和中文。后来，随着社区推出了更多功能更完善的配装工具，我便停止了对该项目的维护。
